# Milestone Summary Template

Generates the summary document when a milestone is completed.

## Frontmatter Schema

```yaml
---
type: milestone-summary
milestone: "{milestone_id}"
milestone_name: "{name}"
status: "complete"
created: "{iso_timestamp}"
phases_completed: "{count}"
total_duration: "{duration}"
---
```

## Generation Triggers

```yaml
triggers:
  automatic:
    condition: "Last phase of milestone verified"
    workflow: "workflows/transition.md"
    
  manual:
    command: "/idumb:complete-milestone"
    requires: "All phases verified"
```
